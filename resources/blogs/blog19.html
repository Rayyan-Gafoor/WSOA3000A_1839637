Discover The Cosmos

Hi there my fellow weirdos, Im sure you know my name, Zariyah, but I was recently told to always state who I am. Apperantly it will help me in my journey to enlightenment. Please dont ask me why, because like you I'm lost and confused. Anyway lets see what we have today:
So as a web designer I wanted to implement something interesting to my website, something that everyone can appreciate. As you have noticed in my latest blogs, I've been captivated by the idea of the Cosmos. As a person, I suffer from not knowing my place in the universe, what my purpose is. For awhile I considered myself to be lost, and so I wanted to create something for people like me. People that i looking to explore the unkown, to cross boundaries that mere mortals can not. As an attempt to ahieve this, I implemented a basic NASA API: APOD (Astronomy Picture of the Day).
Lets take a look at the details further below:

So for those of you that are still confused (don't worry about it, I might be just as confused as you when it comes to this), lets take a closer look at what an API is. According to my understanding an API (application program interface) is a set of tools and protocols used to build applications using digital communication to transfer and collect data securely across a variety of platforms. They help improve our websites and other applications, while getting rid of all those unnecessary steps involed in the developing process. In fact with just the push of a button we have the ability to do this. When we do this not only are we making the site more interesting, but we are also creating a better and more meaningful user experience.

What is the NASA APOD API:
The API is used to return images and descriptions for those images. A new image is taken everyday and thus the name Astronomy Picture of the Day. It does this by allowing the developer to access imagery and metadata, which is accessed through using an API key. See the website below on what the API is :

    Technical implementation

    HTML implementation



    Using a basic implementation of html, a user input is created to receive the date input in the form : YYYY-MM-DD
    Creating a search button is then use to initiate the javascript function.
    Using the img and div elements, an ID is used to identify each. These are the placeholders for where the data being received need to be stored

    Javascript implementation



    The code starts off by creating an Event Listener, which is initiated as soon as the button implemented in the html file is "clicked". When this happen, the input given by the user is then stored in the variable "userDate". This variable is then used in the function "sendApiRequest"
    The sendApiRequest function is used to request the data needed using the Api key, and then appending the stored "userDate" variable at the end of the API key. Once this data is then received it is stored in the variable "data".
    The "data" variable is used in the UseApiData function. This is basically used to display the data we have recieved onto our website. In our case the data is the image source accompanied by the explanation for the image.
